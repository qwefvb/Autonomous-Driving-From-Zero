# 从零开始学习自动驾驶

欢迎来到 **自动驾驶从零开始** 教学项目！这是一个专门针对初学者打造的从零基础入门自动驾驶领域的学习指南。

本人目前为研一在读研究生，在入门自动驾驶的这一年多来走了很多弯路，也积攒了一些经验。考虑到目前市面上还没有一个完整系统的、从零开始的自动驾驶入门教程，故编写本项目供各位小白同学参考。如果你也喜欢我的项目，请送上一个**小小的Star🌟**。

受到本人水平的限制，项目难免有纰漏与错误之处，恳请大家批评指正。目前项目还在不断更新迭代中，预期于2026年7月前完成全部章节的编写。也非常欢迎大家参与到自己细分研究方向的编写中来。

相关更新进度请参考：[更新日志](docs/更新日志.md)

本项目旨在以通俗易懂的语言，辅以经典的学术工作、开源项目及优秀的博客文章，为您构建一个全面、清晰的自动驾驶知识体系。希望这能给你带来帮助！
