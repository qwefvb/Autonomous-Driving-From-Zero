## 📖 关于本项目

> **3 分钟了解为什么你需要这份指南**

### 😰 你是否正在经历这些痛点？

- ❌ **资料零散**：网上博文很多，但看完依然不知道自动驾驶的全貌是什么。
- ❌ **环境折磨**：Ubuntu、CUDA、Docker... 配置环境花掉了一半的科研时间。
- ❌ **论文迷茫**：读了无数篇深度学习论文，却不知道它在自动驾驶中到底怎么用。
- ❌ **复现困难**：GitHub 上的 SOTA 代码跑不通，报错调到怀疑人生。
- ❌ **缺乏体系**：只会调包，不懂底层逻辑，写论文时不知如何创新。

**本项目是什么？**

> **自动驾驶入门指南 | 科研教学向 | 环境感知 | 路径规划 | 端到端算法 | 视觉大模型**

一份系统化、学术导向的学习指南，涵盖：

- **全栈技术**：从底层传感器数据处理到高层决策规划。
- **学术深度**：深入解析 BEV, Occupancy, World Models 等前沿技术栈。
- **科研辅助**：分享 Vibe Coding 时代的科研提效工具与工作流。
- **教学导向**：以通俗易懂的语言解释复杂的学术概念。

### 1. 深度学习基础

自动驾驶算法的灵魂。本节介绍最核心的基础概念，为后续研究打下理论支撑。

- [1.1 神经网络核心概念](docs/01_深度学习基础/1.1_神经网络核心概念.md)
  从感知机、激活函数到前向传播与反向传播，搭建理解深度学习模型的最小知识框架。
- [1.2 卷积神经网络 (CNN)](docs/01_深度学习基础/1.2_卷积神经网络CNN.md)
  理解卷积如何提取局部空间特征，以及 CNN 为什么长期是视觉感知任务的基础骨架。
- [1.3 Transformer 机制](docs/01_深度学习基础/1.3_Transformer机制.md)
  学习注意力机制、多头注意力与编码结构，为理解 BEV、VLM 和大模型方法打好基础。

### 2. 自动驾驶环境感知算法

环境感知是自动驾驶系统的“眼睛”。这一部分会从二维视觉任务一路走到三维理解、多目标跟踪与轨迹预测，帮助你建立对感知任务全链路的整体认识。

- **2.1 2D 感知：检测、车道线与分割**
  先从最经典的图像级任务入手，理解车辆、行人、道路边界与可行驶区域是如何被识别出来的。
  - [2.1.1 2D 目标检测](docs/02_自动驾驶环境感知算法/2.1.1_2D目标检测.md)
    学习如何在图像中定位和分类车辆、行人等关键目标，这是自动驾驶视觉感知的入门任务。
  - [2.1.2 2D 车道线检测](docs/02_自动驾驶环境感知算法/2.1.2_2D车道线检测.md)
    聚焦道路结构理解，认识车道线在横向控制、导航约束与安全驾驶中的作用。
  - [2.1.3 2D 图像分割](docs/02_自动驾驶环境感知算法/2.1.3_2D图像分割.md)
    从像素级角度理解场景，把道路、车辆、天空和行人区域更细致地划分出来。
- **2.2 3D 感知：目标检测、BEV、建图与 Occupancy**
  从单帧图像走向空间表达，理解纯视觉、点云和多模态方案如何恢复真实世界的三维结构。
  - [2.2.1 3D 目标检测（纯视觉）](docs/02_自动驾驶环境感知算法/2.2.1_3D目标检测-纯视觉.md)
    关注仅依赖摄像头时如何估计目标的三维位置与尺寸，理解纯视觉路线的难点与潜力。
  - [2.2.2 3D 目标检测（Lidar 点云）](docs/02_自动驾驶环境感知算法/2.2.2_3D目标检测-lidar点云.md)
    学习如何直接利用点云几何结构进行三维检测，认识激光雷达在空间感知中的优势。
  - [2.2.3 3D 目标检测（多模态融合）](docs/02_自动驾驶环境感知算法/2.2.3_3D目标检测-多模态融合.md)
    理解图像语义与点云几何如何互补，提升复杂场景下三维检测的准确性与鲁棒性。
  - [2.2.4 在线建图 (Online Mapping)](docs/02_自动驾驶环境感知算法/2.2.4_在线建图.md)
    介绍车辆在行驶过程中如何持续构建道路拓扑与语义地图，为高阶驾驶提供结构化先验。
  - [2.2.5 3D Occupancy (占用栅格)](docs/02_自动驾驶环境感知算法/2.2.5_3D_Occupancy.md)
    从更稠密的空间表达视角理解场景，认识 Occupancy 在统一感知与规划中的价值。
- [2.3 深度估计与3D重建](docs/02_自动驾驶环境感知算法/2.3_深度估计与3D重建.md)
  关注如何从图像中恢复深度信息，为三维场景理解与重建提供基础能力。
- [2.4 目标跟踪算法](docs/02_自动驾驶环境感知算法/2.4_目标跟踪算法.md)
  解决“看见一次”到“持续看见”的问题，让系统具备稳定理解动态交通参与者的能力。
- [2.5 轨迹预测算法](docs/02_自动驾驶环境感知算法/2.5_轨迹预测算法.md)
  进一步从“当前状态”推演到“未来行为”，为规划决策提供先验信息。
- [2.6 开放场景理解与异常检测](docs/02_自动驾驶环境感知算法/2.6_其他感知任务.md)
  面向长尾与未知风险场景，理解系统如何发现分布外目标、异常事件与潜在危险。

### 3. 自动驾驶规划算法

规划模块负责回答“下一步该怎么走”。本节会从可解释的规则型方法出发，再过渡到数据驱动的学习型范式，帮助你理解自动驾驶决策的核心逻辑。

- [3.1 规则型算法](docs/03_自动驾驶规划算法/3.1_规则型算法.md)
  学习传统自动驾驶系统中最常见的规划框架，理解搜索、采样、代价函数与约束设计。
- [3.2 学习型算法](docs/03_自动驾驶规划算法/3.2_学习型算法.md)
  关注强化学习、模仿学习等新路线，理解数据驱动方法如何直接学习驾驶策略。

### 4. 端到端自动驾驶算法

端到端方法试图打通“感知-预测-规划”之间的模块边界，是当前学术界和工业界都高度关注的方向。本节会带你从基础范式一路走向世界模型与 VLA 融合的最新趋势。

- [4.1 传统端到端](docs/04_端到端自动驾驶算法/4.1_端到端.md)
  了解早期端到端方法如何直接从传感器输入预测控制量或轨迹，以及它们的优势与争议。
- [4.2 世界模型](docs/04_端到端自动驾驶算法/4.2_世界模型.md)
  理解模型如何在隐空间中“想象未来”，并借此完成预测、规划与决策。
- [4.3 视觉-语言-动作模型 (VLA)](docs/04_端到端自动驾驶算法/4.3_VLA.md)
  关注语言先验如何进入驾驶系统，让模型具备更强的语义理解与任务泛化能力。
- [4.4 世界模型与VLA融合范式](docs/04_端到端自动驾驶算法/4.4_世界模型与VLA融合范式.md)
  面向更前沿的统一范式，探索生成式建模、多模态理解与驾驶动作控制的融合路径。

### 5. 大模型基础

掌握大模型在自动驾驶中的核心链路：先理解语言模型，再补齐视觉基础模型，最后走向多模态理解与高效微调落地。

- [5.1 大语言模型（LLM）基础](docs/05_大模型基础/5.1_大语言模型（LLM）基础.md)
  从 `token`、`embedding`、`Transformer` 到 `BERT / GPT`，建立理解 VLM 所需的语言侧基础。
- [5.2 视觉基础大模型](docs/05_大模型基础/5.2_视觉基础大模型.md)
  聚焦 `DINO`、`SAM`、`Grounded SAM` 等视觉基础模型，理解其在特征提取、自动标注与数据闭环中的作用。
- [5.3 视觉语言模型 (VLM)基础](docs/05_大模型基础/5.3_视觉语言模型（VLM）基础.md)
  介绍 `CLIP`等经典路线与`Qwen3 VL`等最先进的VLM模型，理解图像特征如何接入 LLM 并服务场景理解与可解释性。
- [5.4 VLM使用与微调](docs/05_大模型基础/5.4_VLM使用与微调.md)
  覆盖 `LoRA`、`Adapter`、`P-Tuning`、`RLHF / DPO`、量化与推理加速，帮助大模型在自动驾驶任务中高效落地。
