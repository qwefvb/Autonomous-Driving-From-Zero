# 7.1 Vibe Coding 工具

“Vibe Coding” 并不意味着盲目尝试，而是强调通过直觉引导，并利用高效工具快速将想法转化为原型。在自动驾驶这种复杂的系统工程中，合理的工具链能让你事半功倍。

## 1. AI 编程助手

- **Cursor**：目前最推荐的 IDE。它集成了 Claude 3.5 Sonnet 等顶尖模型，能根据你的整个代码库（Codebase）进行对话和重构。
- **GitHub Copilot**：老牌劲旅，其自动补全功能依然是非常强大的。
- **Trae / Windsurf**：新兴的 AI 辅助编程工具，提供了差异化的交互体验。

## 2. 知识管理与文档

- **Notion**：适合整理学习笔记、项目进度和团队协作。
- **Obsidain / Logseq**：如果你喜欢双向链接和本地存储，构建个人的“第二大脑”。
- **Zotero**：文献管理的必备工具，配合插件可以实现论文的自动重命名和云同步。

## 3. 效率提升神器

- **Claude / ChatGPT**：除了写代码，它们是你理解复杂 Arxiv 论文公式、翻译学术词汇的最佳伴侣。
- **Docker**：自动驾驶环境极其复杂，学会使用 Docker 能让你在不同机器上无痛复现代码环境。

---

## 🛠️ 实践建议
1. 尝试使用 Cursor 的 `@Codebase` 功能来理解一个庞大的开源感知项目（如 BEVFormer）。
2. 使用 AI 工具帮你写一个 Python 脚本，自动下载你感兴趣的 Arxiv 领域最新论文。
