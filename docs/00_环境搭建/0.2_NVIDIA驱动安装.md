# 0.2 nvidia-smi 驱动的安装

没有正确安装驱动，你的显卡就只是一个“昂贵的摆设”。

## 1. 检查显卡型号

```bash
lspci | grep -i nvidia
```

## 2. 安装驱动的几种方式

### 方式一：使用 Ubuntu 软件和更新 (推荐新手)
1.  打开 **Software & Updates**。
2.  点击 **Additional Drivers** 选项卡。
3.  选择标记为 `(recommended)` 的驱动版本。
4.  点击 **Apply Changes**，重启电脑。

### 方式二：命令行自动化安装
```bash
sudo ubuntu-drivers autoinstall
sudo reboot
```

## 3. 验证安装

重启后，在终端输入：
```bash
nvidia-smi
```
如果你看到了显卡信息表格（包含显存占用、驱动版本等），说明安装成功！

## 4. 常见问题：循环登录/黑屏

如果安装后无法进入桌面，通常是因为内核版本不匹配或驱动冲突。
- 解决方法：按 `Ctrl + Alt + F2` 进入命令行模式，卸载驱动 `sudo apt purge nvidia*`，尝试重新安装其他版本。
