# 0.5 Anaconda 的安装与使用

Anaconda 是管理 Python 虚拟环境的神器，能有效避免项目间的依赖冲突。

## 1. 安装

建议安装 **Miniconda**（轻量版），仅包含核心功能。
1. 下载脚本：`wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh`
2. 执行安装：`bash Miniconda3-latest-Linux-x86_64.sh`
3. 按照提示输入 `yes` 并重启终端。

## 2. 常用操作

### 环境创建
```bash
conda create -n ad_env python=3.10 -y
```

### 环境切换
```bash
conda activate ad_env
conda deactivate
```

### 包管理
```bash
conda install pytorch torchvision -c pytorch
# 或者使用 pip (在已激活的环境中)
pip install mmcv
```

### 环境清理
```bash
conda env remove -n ad_env
conda clean --all
```

---

## 🛠️ 实践建议
始终保持“一个项目一个环境”的原则。不要直接在 `base` 环境中安装各种杂乱的库。
