# 0.3 Ubuntu 系统常见指令学习

熟练使用命令行是自动驾驶工程师的基本素养。

## 1. 文件与目录操作

- `ls`：列出当前目录文件。 (`ls -l` 查看详情, `ls -a` 查看隐藏文件)
- `cd`：切换目录。 (`cd ..` 返回上一级)
- `mkdir`：创建文件夹。
- `rm`：删除文件或目录。 (**慎用 `rm -rf`**)
- `cp` / `mv`：复制 / 移动文件。

## 2. 权限管理

- `sudo`：以超级用户权限执行命令。
- `chmod`：修改文件权限。 (`chmod +x script.sh` 让脚本可执行)
- `chown`：修改文件所属用户。

## 3. 系统监控

- `top` / `htop`：查看 CPU 和内存占用。
- `df -h`：查看磁盘空间。
- `free -h`：查看内存剩余。
- `ps -aux | grep XXX`：查找进程。

## 4. 网络相关

- `ifconfig` 或 `ip a`：查看 IP 地址。
- `ping`：测试网络连通性。
