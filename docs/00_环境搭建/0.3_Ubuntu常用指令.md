# 0.3 Ubuntu 常用指令全攻略

**有什么不会的指令问AI，它们会教会你所有知识。但一些基础的指令你必须熟练掌握！**

---

## 📂 1. 文件与目录操作 (File & Directory)

这是最基础的操作，如同你在 Windows 中打开文件夹和移动文件。

| 指令    | 说明                                       | 示例                                                             |
| :------ | :----------------------------------------- | :--------------------------------------------------------------- |
| `pwd`   | 显示当前所在路径 (Print Working Directory) | `pwd`                                                            |
| `ls`    | 列出当前目录下的文件                       | `ls -l` (详细信息), `ls -a` (显示隐藏文件)                       |
| `cd`    | 切换目录                                   | `cd ..` (上级目录), `cd ~` (家目录), `cd /` (根目录)             |
| `mkdir` | 创建文件夹                                 | `mkdir -p scripts/tools` (递归创建)                              |
| `touch` | 创建空白文件                               | `touch config.yaml`                                              |
| `cp`    | 复制文件或目录                             | `cp file.txt copy.txt`, `cp -r folder/ backup/` (-r 用于目录)    |
| `mv`    | 移动文件或重命名                           | `mv old.txt new.txt` (重命名), `mv file.txt ~/Desktop/`          |
| `rm`    | 删除文件或目录                             | `rm file.txt`, `rm -rf folder/` (**慎用：强制删除目录且不提示**) |

> [!CAUTION]
> **关于 `rm -rf` 的警告**
> `rm -rf /` 会删除你整个系统。执行任何带 `-rf` 的删除操作前，请务必肉眼确认路径三次！

## 📝 2. 文件内容查看与编辑

在 Linux 中，“一切皆文件”。掌握如何查看和修改文件内容至关重要。

### 🔍 查看内容

- **`cat`**: 一次性显示整个文件内容。适用于短小文件。
  - 示例: `cat /etc/issue` (查看系统版本)
- **`less`**: 分页查看长文件。按 `Space` 翻页，按 `q` 退出。
  - 示例: `less large_log.log`
- **`head` / `tail`**: 查看文件的前几行或后几行。
  - 示例: `tail -f /var/log/syslog` (**非常有用**：实时追踪日志更新)
- **`grep`**: 在文件中搜索特定文本。
  - 示例: `grep "error" output.log` (寻找包含 error 的行)

### ✍️ 终端编辑器

- **`nano`**: 简单易用，适合新手。
  - 示例: `nano test.py` (Ctrl+O 保存, Ctrl+X 退出)
- **`vim`**: 强大但有门槛。自动驾驶工程师进阶必修。
  - 示例: `vi setup.sh` (按 `i` 进入插入模式, `Esc` 后输入 `:wq` 保存退出)

## 🔐 3. 用户权限与管理

Linux 是多用户系统，权限控制极其严格。

- **`sudo`**: 以超级用户(Root)权限执行命令。
  - 示例: `sudo apt update`
- **`chmod`**: 修改文件权限。
  - 示例: `chmod +x script.py` (赋予可执行权限)
  - 示例: `chmod 777 folder/` (赋予所有人读写执行权限，**生产环境慎用**)
- **`chown`**: 修改文件所属用户。
  - 示例: `sudo chown -R user:user my_project/`
- **`passwd`**: 修改当前用户密码。

## 🚀 4. 系统监控与进程管理

实时了解你的机器在跑什么，资源占用了多少。

- **`htop`**: 交互式进程查看器（比自带的 `top` 更直观美观）。
  - 安装: `sudo apt install htop`
- **`df -h`**: 查看磁盘空间占用情况。
- **`free -h`**: 查看内存（RAM）使用情况。
- **`ps -aux`**: 查看系统中所有正在运行的进程。
  - 配合 grep 使用: `ps -aux | grep ros`
- **`kill`**: 终止进程。
  - 示例: `kill -9 <PID>` (强制杀死进程)
- **`nvidia-smi`**: 查看显卡（GPU）状态、显存占用和驱动版本。

## 🌐 5. 网络操作

用于下载、传输文件以及远程调试。

- **`ifconfig` / `ip a`**: 查看本机的 IP 地址。
- **`ping`**: 测试网络连通性。
  - 示例: `ping www.google.com`
- **`ssh`**: 远程登录到另一台机器。
  - 示例: `ssh username@192.168.1.100`
- **`scp`**: 通过网络安全地复制文件。
  - 示例: `scp file.txt user@remote:/path/to/dest`
- **`wget` / `curl`**: 从网络下载文件。
  - 示例: `wget https://example.com/data.zip`

## 📦 6. 软件包管理 (APT)

Ubuntu 管理软件最常用的工具。

- `sudo apt update`: 更新软件包列表。
- `sudo apt upgrade`: 升级已安装的所有软件。
- `sudo apt install <package>`: 安装软件（如 `sudo apt install git`）。
- `sudo apt remove <package>`: 卸载软件。
- `apt search <keyword>`: 搜索软件包。

## 💡 7. 效率进阶小技巧

### ⌨️ 快捷键

- **`Tab`**: **神级按键**。输入命令或路径的前几个字母后按 Tab，系统会自动补全。按两次会列出所有匹配项。
- **`Ctrl + C`**: 终止当前运行的进程（救命快捷键）。
- **`Ctrl + L`**: 清除屏幕（等同于输入 `clear`）。
- **`Ctrl + R`**: 搜索历史命令。
- **`方向键上/下`**: 快速切换之前输入过的命令。

### 🔗 管道符与重定向

- **`|` (管道)**: 将前一个命令的输出作为后一个命令的输入。
  - 示例: `ls -l | grep ".py"` (列出所有 .py 文件)
- **`>` (重定向)**: 将输出写入文件（覆盖）。
  - 示例: `ls > files_list.txt`
- **`>>` (追加重定向)**: 将输出追加到文件末尾。
