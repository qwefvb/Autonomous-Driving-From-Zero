# 0.1 Ubuntu 系统的安装

## 🌟 为什么选择 Ubuntu？

Ubuntu 是一款基于 Debian 的开源 Linux 发行版，由 Canonical 公司发布。在自动驾驶和人工智能开发领域，Ubuntu 几乎是**事实上的标准操作系统**。

- **卓越的生态支持**：绝大多数深度学习框架（如 PyTorch, TensorFlow）和自动驾驶中间件（如 ROS/ROS2, Apollo, Autoware）都是优先在 Ubuntu 上开发和测试的。
- **硬件驱动友好**：NVIDIA 驱动和 CUDA 环境在 Ubuntu 上的配置最为成熟，能够完美榨取 GPU 的算力。
- **高效的命令行工具**：强大的终端支持，配合 `apt` 包管理器，使得环境搭建极其高效。

> [!IMPORTANT]
> 对于自动驾驶开发，强烈建议安装 **Ubuntu 20.04 LTS** 或 **22.04 LTS** 及以上版本（**不建议安装最新版本，因为最新版本可能存在一些兼容性问题**）。LTS（Long Term Support）版本提供长期的维护支持，稳定性极高。

## 🛠️ 1. 安装方式选择

根据你的硬件条件和使用习惯，选择最适合的安装方式。

### A. 双系统 (Dual Boot) —— 🏆 开发者首选

在物理机硬盘上划出空间直接安装 Ubuntu。

- **推荐理由**：性能无损，可直接调用物理 GPU，是深度学习训练的最佳选择。
- **潜在风险**：安装过程涉及磁盘分区，需谨慎操作以防误删 Windows 数据。
- **📖 延伸阅读：**
  - [Ubuntu20.04 双系统安装详解（一文通关！）](https://blog.csdn.net/wyr1849089774/article/details/133387874)
  - [2020 最详细安装 Ubuntu 指南](https://zhuanlan.zhihu.com/p/135953477)
  - [Windows + Ubuntu 22.04 双系统安装全过程](https://blog.csdn.net/hclam/article/details/148404947)
  - [win10+Ubuntu20.4双系统安装详细教程（包含bios设置）](https://blog.csdn.net/m0_37860943/article/details/128061708)

### B. WSL2 (Windows Subsystem for Linux) —— 💻 Windows 用户折中方案

Windows 内置的 Linux 子系统，支持直接在 Windows 环境下运行 Linux 应用。

- **推荐理由**：无需重启系统，文件互通极佳，目前已支持 GPU 调用，性能接近原生。
- **局限性**：在处理底层硬件通信（如特定的串口、网口调试）时可能存在配置成本。
- **📖 延伸阅读：**
  - [WSL2 + Ubuntu20.04 全流程安装指南（支持安装至 D 盘）](https://blog.csdn.net/natsuago/article/details/145594631)
  - [Windows 10/11 WSL2 完整安装与使用教程](https://blog.csdn.net/weixin_44818765/article/details/157736533)

### C. 虚拟机 (VMware/VirtualBox) —— 🧪 仅推荐练习

在虚拟机软件中安装 Linux。

- **推荐理由**：绝对安全，适合初学者熟悉 Ubuntu 命令行。
- **局限性**：**无法直接使用 NVIDIA 显卡**进行深度学习训练，性能损耗较大。
- **📖 延伸阅读：**
  - [VMware 虚拟机安装 Ubuntu 教程 (超详细)](https://blog.csdn.net/weixin_52799373/article/details/139055794)
  - [Ubuntu 虚拟机的安装教程](https://blog.csdn.net/2502_90817102/article/details/147247894)

## 📋 2. 准备工作

1.  **下载镜像**：前往 [Ubuntu 官网](https://ubuntu.com/download/desktop) 下载 ISO 文件。
2.  **准备 U 盘**：建议容量 $\ge$ 16GB。
3.  **制作启动盘**：
    - **Windows**: 推荐使用 [Rufus](https://rufus.ie/)。
    - **跨平台**: 推荐使用 [BalenaEtcher](https://www.balena.io/etcher/)。

---

## 🚀 3. 安装步骤要点 (物理机安装)

> [!CAUTION]
> **数据安全**：在进行分区操作前，请务必备份重要数据！

1.  **引导启动**：插入 U 盘，重启电脑，连按 `F12` 或 `F2`（根据主板品牌）进入 BIOS/Boot Menu，选择从 U 盘启动。
2.  **进入安装器**：选择 **Try or Install Ubuntu**。
3.  **分区建议 (针对 512GB+ 硬盘)**：
    - `/` (根目录)：**100GB+**。系统核心文件。
    - `/home`：**300GB+**。存放数据集、代码和模型权重，建议预留最大空间。
    - `swap` (交换分区)：建议为物理内存的 **1-2 倍**（如 16GB 内存配 16-32GB swap）。

## 🪄 4. 安装后首要任务

安装完成后，首先需要更换镜像源（默认国外源下载速度较慢）。

> [!TIP]
> 建议更换为国内主流镜像站（如清华、阿里、中科大）。
> **参考教程**：[Ubuntu 系统更换镜像源详细步骤](https://blog.csdn.net/m0_58963318/article/details/148364245)

打开终端 (`Ctrl + Alt + T`)，依次运行以下命令：

```bash
# 更新软件包索引
sudo apt update

# 升级所有已安装的软件包到最新版本
sudo apt upgrade
```
