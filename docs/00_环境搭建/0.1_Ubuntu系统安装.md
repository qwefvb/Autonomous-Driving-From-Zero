# 0.1 Ubuntu 系统的安装

Ubuntu 是自动驾驶开发的首选操作系统，建议安装 **Ubuntu 22.04 LTS** 或 **20.04 LTS**。

## 1. 安装方式选择

- **双系统 (Dual Boot)**：在物理机上直接安装。推荐，因为自动驾驶仿真和深度学习需要直接访问显卡硬件，性能最好。
- **虚拟机 (VMware/VirtualBox)**：适合初次尝试，但无法使用 NVIDIA 显卡进行深度学习训练。
- **WSL2 (Windows Subsystem for Linux)**：Windows 用户的折中方案，性能接近原生，支持 GPU。

## 2. 准备工作

1.  **下载镜像**：前往 [Ubuntu 官网](https://ubuntu.com/download/desktop) 下载 ISO 文件。
2.  **制作启动盘**：使用 **Rufus** (Windows) 或 **BalenaEtcher** (Cross-platform) 将镜像写入 U 盘。

## 3. 安装步骤要点

1.  从 U 盘启动（通常连按 F12 或 F2 进入 BIOS）。
2.  选择 **Try or Install Ubuntu**。
3.  **分区建议**：
    *   `/` (根目录)：建议 100GB+。
    *   `/home`：存放个人数据，建议预留大空间。
    *   `swap` (交换分区)：建议为内存大小的 1-2 倍。

## 4. 安装后首要任务

安装完成后，首先运行以下命令更新源：
```bash
sudo apt update
sudo apt upgrade
```
