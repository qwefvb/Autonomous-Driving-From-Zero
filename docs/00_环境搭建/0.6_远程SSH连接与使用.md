# 0.6 远程 SSH 的连接与使用

在实际工作中，你通常会连接实验室或公司的强大服务器进行训练，而不是直接在本地机器上运行。

## 1. 基础连接

```bash
ssh username@server_ip
```

## 2. 免密登录 (配置 SSH Key)

1. **本地生成密钥**：`ssh-keygen -t rsa`
2. **拷贝公钥到服务器**：`ssh-copy-id username@server_ip`
3. 之后再登录就不需要输入密码了。

## 3. 使用 VS Code 进行远程开发

这是目前最优雅的开发方式：
1. 在本地安装 **VS Code**。
2. 安装扩展 **Remote - SSH**。
3. 点击左下角的“远程窗口”图标，连接到你的服务器。
4. 你可以像编辑本地代码一样编辑服务器上的代码，并直接在 VS Code 终端运行程序。

## 4. 后台运行任务 (tmux)

如果你直接通过 SSH 运行代码，一旦电脑断网或关机，任务就会终止。
- 使用 **tmux** 或 **screen**：
  ```bash
  tmux new -s train_session  # 创建新会话
  # 运行训练脚本...
  # Ctrl+B 然后按 D 退出会话 (任务继续运行)
  tmux attach -t train_session # 重新连接会话
  ```
