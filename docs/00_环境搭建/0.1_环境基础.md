# 0. 环境搭建

本章将带你从零开始搭建自动驾驶算法开发的基础环境。我们将主要使用 **Ubuntu** 操作系统和 **Anaconda** 进行环境隔离与管理。

## 0.1 Ubuntu 系统教学

自动驾驶领域的大多数开源项目（如 ROS, Apollo, Autoware）和深度学习框架（PyTorch, TensorFlow）在 Linux 系统下支持最为完美。

### 为什么选择 Ubuntu？
- **稳定性**：适合长期运行计算密集型任务。
- **社区支持**：遇到问题极易通过搜索找到解决方案。
- **工具链丰富**：自带或极易安装开发所需的各种编译器和库。

### 基础操作指南
- **终端 (Terminal)**：这是你的主战场。通过 `Ctrl + Alt + T` 唤起。
- **常用命令**：
  - `sudo apt update && sudo apt upgrade`：更新系统软件包。
  - `ls`, `cd`, `mkdir`, `rm`：文件系统管理。
  - `top` / `htop`：监控系统资源。

## 0.2 Anaconda 环境管理

在深度学习开发中，不同的项目往往依赖不同版本的 Python 或库（例如项目 A 需要 PyTorch 1.7，项目 B 需要 PyTorch 2.0）。Anaconda 允许我们为每个项目创建独立的“虚拟环境”，互不干扰。

### 安装 Anaconda/Miniconda
建议在 Ubuntu 下使用命令行安装：
1. 下载安装脚本：`wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh`
2. 运行安装：`bash Miniconda3-latest-Linux-x86_64.sh`
3. 激活环境：`source ~/.bashrc`

### 常用环境管理命令
- **创建环境**：`conda create -n ad_zero python=3.8`
- **激活环境**：`conda activate ad_zero`
- **安装库**：`pip install torch torchvision` 或 `conda install -c pytorch pytorch`
- **导出环境**：`conda env export > environment.yml`
