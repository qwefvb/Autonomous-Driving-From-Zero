# 0.4 科学上网

在自动驾驶开发中，我们需要频繁从 GitHub、Hugging Face 等国外服务器下载代码和权重，一个稳定的网络环境必不可少。

## 1. 为什么需要？

- **GitHub 克隆速度慢**：直接克隆可能只有几 KB/s。
- **pip/conda 安装失败**：许多深度学习库的源都在海外。
- **访问文档**：如 PyTorch 官方文档等。

## 2. 常用方案

- **Clash for Windows / Verge (Linux 版本)**：目前最主流的图形化代理工具。
- **CFW 的 TUN 模式**：接管整个系统的流量，配置最省心。
- **终端代理配置**：
  如果你已经运行了代理软件（假设端口为 7890），可以在终端临时配置：
  ```bash
  export http_proxy="http://127.0.0.1:7890"
  export https_proxy="http://127.0.0.1:7890"
  ```

## 3. 换源 (作为补充)

如果网络实在不给力，可以考虑将软件源更换为国内镜像：
- **apt 换源**：推荐清华源或阿里源。
- **pip 换源**：`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
- **conda 换源**：修改 `~/.condarc`。
