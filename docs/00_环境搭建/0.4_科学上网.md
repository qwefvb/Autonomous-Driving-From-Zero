# 0.4 科学上网与网络优化指南

在自动驾驶研究与开发中，一个稳定、高速的网络环境是提高效率的第一步。我们需要频繁地从 GitHub 克隆代码、从 Hugging Face 下载大模型权重，或者拉取 Docker 镜像。

## 🛠️ 1. 核心方案选择

### 🖥️ 图形化界面 (GUI) - 推荐新手使用

如果你使用的是 Ubuntu 桌面版，图形化工具是最直观的选择。

- **Clash Verge Rev**: 目前 Linux 上最推荐的 Clash 客户端，支持中文，界面现代化。
  - **GitHub**: [clash-verge-rev/clash-verge-rev](https://github.com/clash-verge-rev/clash-verge-rev)
  - **特点**: 图形化界面好，小白易上手

### ⌨️ 命令行工具 (CLI) - 进阶必备

在服务器或没有图形界面的环境下，我们需要命令行代理。

- **Clash for Linux**: 专门为 Linux 定制的命令行脚本。
  - **GitHub**: [wnlen/clash-for-linux](https://github.com/wnlen/clash-for-linux)
  - **特点**: 简约干净，命令行操作，你值得拥有

## 💻 2. 终端代理配置

即便开启了代理软件，Linux 终端默认往往是不走代理的。你需要手动告诉终端去哪里找代理。

### 🕒 临时生效 (当前终端窗口)

```bash
export http_proxy="http://127.0.0.1:7890"
export https_proxy="http://127.0.0.1:7890"
```

### ♾️ 永久生效 (推荐)

将代理配置写入 Shell 配置文件（如 `~/.bashrc` 或 `~/.zshrc`），并设置开关快捷键。

1. 编辑配置文件: `nano ~/.bashrc`
2. 在文件末尾添加以下内容：

```bash
# Proxy aliases
alias proxy_on='export http_proxy=http://127.0.0.1:7890; export https_proxy=http://127.0.0.1:7890; echo "Proxy On"'
alias proxy_off='unset http_proxy; unset https_proxy; echo "Proxy Off"'
```

3. 生效配置: `source ~/.bashrc`
4. 之后只需输入 `proxy_on` 即可开启。

## 🐙 3. Git 代理配置 (关键)

很多时候终端配置了代理，但 `git clone` 依然很慢。这是因为 Git 可以独立设置代理。

```bash
# 设置代理 (假设端口 7890)
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890

# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
```

## 🐳 4. Docker 代理配置

Docker 容器拉取镜像时不受系统环境变量影响，需要单独配置。

1. 创建目录: `sudo mkdir -p /etc/systemd/system/docker.service.d`
2. 创建文件: `sudo nano /etc/systemd/system/docker.service.d/http-proxy.conf`
3. 写入内容:

```ini
[Service]
Environment="HTTP_PROXY=http://127.0.0.1:7890"
Environment="HTTPS_PROXY=http://127.0.0.1:7890"
```

4. 重启 Docker:

```bash
sudo systemctl daemon-reload
sudo systemctl restart docker
```

## 🔄 5. 国内镜像源 (Plan B)

如果网络环境确实极差，可以作为补充方案。

- **APT 换源**: 使用清华大学或阿里云镜像站。
- **pip 换源**:
  ```bash
  pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  ```
- **Hugging Face 镜像**:
  ```bash
  export HF_ENDPOINT="https://hf-mirror.com"
  ```
