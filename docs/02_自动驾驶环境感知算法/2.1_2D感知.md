# 2.1 2D感知：目标检测、车道线检测、图像分割

2D 图像感知是自动驾驶最基础的环节，虽然现在行业重心在向 3D/BEV 转移，但 2D 感知的理论是所有后续算法的基石。

## 1. 2D 目标检测 (Object Detection)

目标检测的任务是在图像中找出所有感兴趣的物体，并给出它们的类别和边界框 (Bounding Box)。

### 经典算法
- **两阶段 (Two-Stage)**：以 **Faster R-CNN** 为代表，先生成候选区域，再进行分类和回归。精度高但速度慢。
- **单阶段 (One-Stage)**：以 **YOLO (You Only Look Once)** 系列和 **SSD** 为代表。将检测视为单一回归问题，速度极快，是量产方案的首选。
- **基于 Transformer**：**DETR** 引入了 End-to-End 的检测思路，去除了 NMS 等后处理。

### 推荐学习项目
- [MMDetection](https://github.com/open-mmlab/mmdetection)：OpenMMLab 出品的检测工具箱，包含了几乎所有主流算法。

## 2. 2D 车道线检测 (Lane Detection)

车道线检测对于车辆的横向控制（如 LKA 车道保持）至关重要。

### 主要挑战
- 线条细长、遮挡严重。
- 光照变化、雨雪天气干扰。

### 经典工作
- **SCNN (Spatial CNN)**：利用像素间的空间关系。
- **LaneATT**：基于 Anchor 的快速检测方法。
- **CLRNet**：目前性能顶尖的方案之一。

---

## 3. 2D 图像分割 (Image Segmentation)

分割是对图像中的每一个像素进行分类，能够提供比边界框更细致的语义信息。

### 分类
- **语义分割 (Semantic Segmentation)**：区分背景、道路、车辆等类别（不区分个体）。
- **实例分割 (Instance Segmentation)**：不仅区分类别，还区分同一个类别的不同个体（如车 A 和车 B）。

### 经典模型
- **U-Net**：结构简洁，广泛应用于医疗和通用分割。
- **DeepLabV3+**：空洞卷积的巅峰之作。
- **Segment Anything (SAM)**：Meta 推出的万物皆可分割大模型，正在改变分割任务的范式。
