# 2.5 轨迹预测算法

预测是连接感知与规划的纽带。我们需要预测周围交通参与者（车、人、骑行者）在未来数秒内的可能路径。

## 1. 核心挑战
- **多模态性 (Multi-modality)**：一个司机在路口既可能直行也可能转弯，预测算法需要给出多种可能的轨迹。
- **交互性 (Interactivity)**：我的动作会影响别人的动作。

## 2. 经典模型
- **VectorNet**：利用向量化表示（Vectorized Representation）处理地图和轨迹特征。
- **TNT (Target-driveN Trajectory prediction)**：先预测目标点，再补全轨迹。
- **Waymo Motion Prediction 挑战赛冠军方案**：通常采用层级化的 Transformer 结构。
