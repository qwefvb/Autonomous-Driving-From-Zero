# 2.6 其他感知任务

除了主流的 3D 检测和分割，自动驾驶感知系统还包含许多进阶任务，用于提升系统的鲁棒性和对复杂场景的理解能力。

## 1. 场景理解与全景分割 (Panoptic Segmentation)

全景分割结合了**语义分割**（区分道路、建筑等背景）和**实例分割**（区分每一辆车、每一个行人）。

- **核心挑战**：如何高效地处理重叠的实例以及保持时序一致性。
- **代表工作**：Panoptic SegNet, Panoptic-DeepLab。

## 2. 开放词汇感知 (Open Vocabulary Perception)

传统的感知模型只能识别训练集中预定义的类别（如“车”、“行人”）。

- **为什么需要**：现实世界中存在无数长尾类别（如“路边的垃圾桶”、“特殊的施工标志”）。
- **技术路径**：利用 CLIP 等视觉-语言预训练大模型，将视觉特征与自然语言描述对齐。
- **代表工作**：Open-Vocabulary DETR, Grounding DINO。

## 3. 异常检测 (Anomaly Detection / OOD)

识别场景中的“未知物” (Out-of-Distribution, OOD)，即模型从未见过的物体。

- **重要性**：对于安全至关重要，当遇到奇形怪状的物体时，系统应能及时告警或采取保守策略。
- **方法**：不确定性估计 (Uncertainty Estimation)、重建误差法。
