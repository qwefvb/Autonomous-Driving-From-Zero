# 2.1.2 2D 车道线检测

车道线检测决定了车辆“该走哪条道”，是横向控制的基础。

## 1. 传统方法 vs 深度学习

- **传统方法**：基于霍夫变换 (Hough Transform) 或边缘提取。容易受到光照和路面杂质干扰。
- **深度学习方法**：将车道线视为像素级分割或基于 Anchor 的回归任务。

## 2. 经典模型

- **LaneATT**：基于 Anchor 机制，兼顾了精度和极高的推理速度。
- **CLRNet**：目前性能顶尖的方案，通过多尺度特征融合提升了检测稳定性。

## 3. 挑战与长尾场景

- 磨损模糊的车道线。
- 雨夜路面反光。
- 复杂路口（分叉、汇入）。
