# 2.3 深度估计与3D重建

深度信息是 2D 图像通往 3D 世界的桥梁。

## 1. 深度估计 (Depth Estimation)
- **单目深度估计**：从单张图片预测深度。
- **自监督学习**：利用视频序列中的时序一致性（如 **Monodepth2**）或立体匹配（双目）来训练，无需昂贵的标注。

## 2. 3D 重建
- **NeRF (Neural Radiance Fields)**：通过神经网络隐式表示场景，可以生成高质量的视角合成，正广泛应用于自动驾驶的数据仿真和闭环测试。
- **Gaussian Splatting**：一种比 NeRF 速度更快的显式场景表示方法，是目前学术界的热点。
