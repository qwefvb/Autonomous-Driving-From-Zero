# 2.1.3 2D 图像分割

分割任务为自动驾驶系统提供了最详尽的视觉语义信息。

## 1. 语义分割 (Semantic Segmentation)

- **目标**：区分道路、人行道、植被、建筑等。
- **经典模型**：DeepLab 系列、SegFormer。

## 2. 实例分割 (Instance Segmentation)

- **目标**：不仅要区分类别，还要区分具体的个体（如第 1 辆车、第 2 辆车）。
- **经典模型**：Mask R-CNN、SOLOv2。

## 3. 分割在自动驾驶中的价值

- **可行驶区域检测**：明确车辆哪些地方能开。
- **精细轮廓**：相比边界框，分割能提供更准确的障碍物边缘，对泊车等精细操作至关重要。
