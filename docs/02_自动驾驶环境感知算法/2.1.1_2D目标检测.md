# 2.1.1 2D 目标检测

目标检测是感知的核心任务，负责识别并定位图像中的物体。

## 1. 经典算法流派

- **YOLO 系列 (One-Stage)**：速度极快，是量产自动驾驶系统的首选。目前已经演进到 YOLOv11。
- **Faster R-CNN (Two-Stage)**：经典的基于候选区域 (Anchor) 的方法。
- **DETR (Transformer-based)**：消除了 NMS（非极大值抑制）等复杂后处理，实现了端到端的检测。

## 2. 核心评估指标

- **mAP (mean Average Precision)**：衡量检测精度最常用的指标。
- **FPS (Frames Per Second)**：衡量推理速度。

## 3. 在自动驾驶中的应用

- 识别前方车辆、行人、骑行者。
- 识别交通标志、红绿灯。
