# 2.4 目标跟踪算法

仅仅检测到物体是不够的，我们还需要知道“当前的这辆车”就是“上一帧的那辆车”，从而获取它的速度、运动轨迹。

## 1. 核心任务 (MOT)
多目标跟踪 (Multi-Object Tracking, MOT) 需要在每一帧中检测出物体，并将它们关联起来形成轨迹。

## 2. 经典算法范式
- **TBD (Tracking-by-Detection)**：先检测，再关联。
  - **SORT / DeepSORT**：利用卡尔曼滤波 (Kalman Filter) 预测位置，通过匈牙利算法 (Hungarian Algorithm) 进行匹配。
- **JDT (Joint Detection and Tracking)**：在一个网络里同时完成检测和跟踪。
  - **ByteTrack**：通过简单且极其高效的关联策略（Byte 数据关联）登顶 SOTA。

## 3. 3D 跟踪
在 BEV 空间或 3D 空间进行跟踪，能够更准确地利用物理世界的运动规律。
