# 1.1.2 卷积神经网络 (CNN)：计算机视觉的灵魂

卷积神经网络（Convolutional Neural Networks, CNN）是深度学习在计算机视觉领域取得突破的核心力量。与传统的全连接神经网络（MLP）相比，CNN 能够有效地捕获图像中的**空间层次结构**。

---

## 1. 为什么需要 CNN？(CNN vs. MLP)

如果我们使用全连接网络处理一张 $1024 \times 1024$ 的彩色图片：

- **参数爆炸**：输入层将有 $1024 \times 1024 \times 3 \approx 300$ 万个节点。如果第一个隐层有 1000 个神经元，仅第一层权重就有 30 亿个！
- **丢失空间信息**：MLP 需要将图片展平（Flatten）为一维向量，这破坏了像素之间的邻近关系。

**CNN 的三大法宝：**

1. **局部感受野 (Local Receptive Fields)**：每个神经元只关注输入图像的一个小区域。
2. **权值共享 (Weight Sharing)**：同一层中的多个神经元可以使用相同的卷积核，极大地减少了参数量。
3. **空间下采样 (Pooling)**：通过聚合信息降低分辨率，增加特征的鲁棒性。

---

## 2. 核心组件详解

### 2.1 卷积运算 (Convolution)

卷积层是 CNN 的核心。它通过一个可学习的**卷积核 (Kernel/Filter)** 在图像上滑动，计算局部加权和。

![卷积层计算原理示意图]

- **步长 (Stride)**：卷积核每次滑动的像素数。步长越大，输出特征图越小。
- **填充 (Padding)**：在输入边缘补零，以控制输出大小并保留边缘信息。

### 2.2 池化层 (Pooling)

池化通常紧跟在卷积层之后，用于减少数据维度。

- **最大池化 (Max Pooling)**：选取区域内的最大值，保留最显著特征。
- **平均池化 (Average Pooling)**：计算区域内的平均值，保留背景信息。

### 2.3 整体架构

典型的 CNN 遵循 `[CONV -> RELU -> POOL] * N -> FC -> Softmax` 的模式。

## 3. 经典模型演进

1.  **LeNet-5 (1998)**：CNN 的开山之作，用于手写数字识别。
2.  **AlexNet (2012)**：开启了深度学习元年，引入了 ReLU 和 Dropout。
3.  **VGG (2014)**：强调“深”的重要性，通过堆叠 3x3 小卷积核构建深层网络。
4.  **ResNet (2015)**：**里程碑式工作**。引入“残差连接”，解决了深层网络训练难的问题。

### 重点：残差学习 (Residual Learning)

在 ResNet 之前，网络越深越难训练（梯度消失/爆炸）。ResNet 提出通过“捷径连接 (Shortcut Connection)”学习残差 $F(x) = H(x) - x$。

![ResNet 残差块示意图]

## 4. CNN 在自动驾驶中的应用

在自动驾驶感知算法中，CNN 是绝对的主力（Backbone）。

![自动驾驶中的 CNN 感知应用]

- **目标检测 (Object Detection)**：识别车辆、行人、交通标志（如 YOLO, Faster R-CNN, SSD）。
- **语义分割 (Semantic Segmentation)**：逐像素分类，区分可行驶区域、车道线（如 DeepLabV3+, Mask R-CNN）。
- **3D 目标检测**：在 PointPillars 等算法中，CNN 被用于处理点云投影后的伪图像。

## 5. 代码实践：PyTorch 实现一个简易 CNN

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super(SimpleCNN, self).__init__()
        # 卷积层：输入 3 通道 (RGB)，输出 16 通道，核大小 3x3
        self.conv1 = nn.Conv2d(3, 16, kernel_size=3, stride=1, padding=1)
        # 池化层：2x2 最大池化
        self.pool = nn.MaxPool2d(kernel_size=2, stride=2)
        # 卷积层：输出 32 通道
        self.conv2 = nn.Conv2d(16, 32, kernel_size=3, stride=1, padding=1)
        # 全连接层
        self.fc1 = nn.Linear(32 * 8 * 8, 128) # 假设输入图片是 32x32
        self.fc2 = nn.Linear(128, num_classes)

    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x))) # 32x32 -> 16x16
        x = self.pool(F.relu(self.conv2(x))) # 16x16 -> 8x8
        x = x.view(-1, 32 * 8 * 8)           # 展平
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 实例化模型
model = SimpleCNN(num_classes=10)
print(model)
```
