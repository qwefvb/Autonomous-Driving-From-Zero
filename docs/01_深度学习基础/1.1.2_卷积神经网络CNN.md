# 1.1.2 卷积神经网络 (CNN)

CNN 是计算机视觉领域的基石，它通过局部感受野和权值共享提取图像特征。

## 1. 核心算子

- **卷积 (Convolution)**：提取局部空间特征。
- **池化 (Pooling)**：降低特征图分辨率，增加感受野。
- **归一化 (Normalization)**：如 Batch Norm，加速训练并提高稳定性。

## 2. 经典模型：ResNet

在自动驾驶中，ResNet 是最常用的 Backbone。
- **残差连接 (Residual Connection)**：通过 `y = f(x) + x` 解决了深层网络的退化问题。
- **在 AD 中的应用**：几乎所有的 2D 检测器和 3D BEV 算法都会使用 ResNet50/101 作为基础特征提取器。
