# 1.1.3 Transformer 机制

Transformer 正在统治自动驾驶领域。

## 1. 注意力机制 (Attention)

- **Self-Attention**：计算序列内部各元素之间的关联。
- **Cross-Attention**：计算两个不同序列之间的关联（如在感知中，Query 与图像特征进行交互）。

## 2. Vision Transformer (ViT)

将图像切分为小块 (Patches)，像处理 Token 一样处理图像块。
- **优点**：能够捕获长距离的依赖关系。
- **代价**：计算量大，对小目标的感知能力在初始阶段弱于 CNN。

## 3. 在自动驾驶中的地位

- **多摄像头融合**：利用 Transformer 将不同视角的图像特征聚合到 BEV 空间。
- **时序建模**：利用时序 Transformer 处理车辆的运动信息。
