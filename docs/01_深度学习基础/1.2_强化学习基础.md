# 1.2 强化学习基础

强化学习 (Reinforcement Learning, RL) 是自动驾驶中决策规划和控制的重要技术手段。与监督学习不同，RL 强调智能体 (Agent) 通过与环境的交互来学习最优策略。

## 1. 核心概念

在强化学习中，你需要理解以下基本元素：
- **智能体 (Agent)**：自动驾驶汽车。
- **环境 (Environment)**：道路、交通参与者、天气等。
- **状态 (State, S)**：传感器数据（摄像头、雷达）、车辆位置、速度等。
- **动作 (Action, A)**：油门、刹车、转向角度。
- **奖励 (Reward, R)**：对动作的评价。例如：安全到达目的地（正奖励）、碰撞（极大负奖励）、超速（小负奖励）。
- **策略 (Policy, π)**：从状态到动作的映射函数。

## 2. 常用算法分类

### 基础算法
- **Q-Learning / DQN**：基于价值 (Value-based) 的方法，适用于离散动作空间。
- **Policy Gradient**：直接对策略进行建模的方法。

### 现代主流算法
- **PPO (Proximal Policy Optimization)**：目前工业界最常用的算法，稳定性高。
- **SAC (Soft Actor-Critic)**：适用于连续动作空间（如转向角），具有极佳的采样效率。

## 3. 在自动驾驶中的应用

1.  **决策规划**：在复杂的路口进行变道、超车决策。
2.  **端到端控制**：直接从传感器输入映射到驾驶控制指令。
3.  **模拟器训练**：利用 CARLA、AirSim 等模拟器生成的海量数据进行预训练。
