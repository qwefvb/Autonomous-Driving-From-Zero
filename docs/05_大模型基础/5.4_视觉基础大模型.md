# 5.4 视觉基础大模型 (Vision Foundation Models)

在自动驾驶迈向大模型时代的进程中，视觉基础大模型 (VFM) 扮演了“底层特征提取器”和“自动标注员”的角色。

## 1. DINO 系列：自监督学习的巅峰

DINO (Distillation with no labels) 证明了通过自监督学习可以获得极强的视觉特征。
- **DINOv2**：Meta 发布的重量级工作。它提取的特征可以直接用于深度估计、语义分割而无需微调，其对物体轮廓的理解能力甚至超越了有监督模型。
- **在 AD 中的作用**：作为通用的 Backbone，提升模型在各种环境下的泛化能力。

## 2. SAM (Segment Anything Model)

SAM 是视觉领域的“ChatGPT”，它实现了对任意物体的零样本分割。
- **核心能力**：通过点、框或文本提示 (Prompt) 实时分割出图像中的任何目标。
- **在 AD 中的作用**：
    - **自动标注**：极大地降低了感知数据的标注成本。
    - **数据清洗**：辅助识别和抠取复杂的长尾场景数据。

## 3. Grounded SAM

结合了 Grounding DINO（目标检测大模型）和 SAM（分割大模型）。
- **流程**：输入文字（如“路边的消火栓”） -> 检测框 -> 分割掩码。
- **意义**：实现了端到端的“文本引导式”感知。

## 4. 基础模型在自动驾驶中的落地路径

1.  **特征增强**：使用 DINOv2 作为感知算法的预训练权重。
2.  **闭环标注 (Auto-labeling)**：利用大模型对海量路测数据进行离线高精度标注，用于训练在线轻量化模型。
3.  **世界模型生成**：利用基础模型的生成能力，构建逼真度极高的仿真场景。
