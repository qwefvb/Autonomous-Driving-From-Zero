# 5.1 自然语言处理 (NLP) 入门

了解自动驾驶的大模型（如 VLM/VLA）之前，必须先掌握 NLP 的核心演进路径。

## 1. 从 Word2Vec 到 RNN
- **Word Embedding**：将单词转化为计算机能理解的向量。
- **RNN / LSTM**：处理序列数据的传统方式，但存在长程依赖丢失的问题。

## 2. Transformer 的崛起
- **Self-Attention**：允许模型同时关注序列中的所有部分。
- **Encoder-Decoder 架构**：机器翻译的标准框架。
