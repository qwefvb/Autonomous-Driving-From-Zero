# 5.3 视觉语言模型 (VLM)

VLM 能够同时理解图像和文字，是自动驾驶感知的“终极形态”。

## 1. 经典工作
- **CLIP (OpenAI)**：将图像和文本对齐到同一个特征空间。
- **BLIP / BLIP-2**：引入了更高效的视觉-语言特征交互。
- **LLaVA**：将视觉特征映射到 LLM 的输入空间，实现“能看会说的 LLM”。

## 2. 在自动驾驶中的应用
- **场景理解**：不仅仅检测到行人，还能理解“这个行人在挥手示意我先行”。
- **可解释性**：让模型解释为什么在某个路口选择减速。
