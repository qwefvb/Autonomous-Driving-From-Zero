# 5.2 大语言模型 (LLM)

LLM 是现代 AI 的引擎，其蕴含的常识和推理能力正在重塑自动驾驶的决策层。

## 1. 核心技术
- **GPT 系列**：生成式预训练 Transformer。
- **RLHF (Reinforcement Learning from Human Feedback)**：通过人类反馈对齐，让模型说话更像人。
- **Chain of Thought (CoT)**：思维链，让模型具备一步步推理的能力。

## 2. 开源力量
- **LLaMA (Meta)**：开启了开源大模型的新时代。
- **Qwen / ChatGLM**：国产优秀的开源大模型。
