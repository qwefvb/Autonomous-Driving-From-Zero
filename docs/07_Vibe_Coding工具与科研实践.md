# 7. Vibe Coding 工具与科研实践

在现代 AI 浪潮下，如何利用工具提升编码和科研效率，是新手与资深开发者的分水岭。

## 1. Vibe Coding 工具
“Vibe Coding” 强调通过直觉和高效工具快速构建原型。
- **GitHub Copilot / Cursor**：AI 辅助编程的巅峰工具，能够帮你快速生成 boilerplate 代码和进行重构。
- **Claude / ChatGPT**：你的私人导师，帮你解释复杂的 Arxiv 论文公式或调试棘手的 CUDA Bug。
- **Notion / Obsidian**：用于构建你的自动驾驶知识库。

## 2. 科研实践建议
- **读论文的技巧**：
  1. 先读 Abstract, Introduction 和结论。
  2. 看图表（图表往往包含模型架构的核心）。
  3. 最后钻研细节。
- **复现实验**：
  1. **Seed 保持一致**：确保结果可重复。
  2. **从最小版本开始**：先在小数据集上跑通 Pipeline。
- **关注 Arxiv**：自动驾驶领域发展极快，每周关注 [Arxiv CV / Robotics](https://arxiv.org/list/cs.CV/recent) 的新文章。
