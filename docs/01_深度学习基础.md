# 1. 深度学习基础

自动驾驶的感知、预测和规划现在几乎都建立在深度学习之上。本章将梳理进入该领域前必须掌握的核心知识。

## 1.1 核心概念与理论

在开始写代码前，你需要理解以下基本概念：
- **感知机与神经网络**：神经元、层、激活函数（ReLU, Sigmoid, Softmax）。
- **反向传播 (Backpropagation)**：梯度下降、链式法则、损失函数（MSE, CrossEntropy）。
- **正则化与优化**：Dropout, Batch Normalization, Adam 优化器。

## 1.2 经典模型演进

### 卷积神经网络 (CNN) - 感知的基石
- **LeNet-5**：手写数字识别的鼻祖。
- **ResNet (He et al., 2015)**：通过残差连接解决了深层网络的退化问题，是目前大多数 2D/3D 感知骨干网络 (Backbone) 的基础。
- **经典工作**：[Deep Residual Learning for Image Recognition](https://arxiv.org/abs/1512.03385)

### Transformer - 现代自动驾驶的核心
- **Attention Is All You Need (Vaswani et al., 2017)**：摒弃了 CNN 和 RNN，完全基于注意力机制。
- **Vision Transformer (ViT)**：将 Transformer 引入视觉领域。
- **在自动驾驶中的地位**：现代算法如 DETR3D, BEVFormer 等无一不依赖 Transformer 进行多摄像头特征融合和时序建模。

## 1.3 经典学习项目与博客

### 推荐项目
- **PyTorch 官方教程**：最推荐的入门起点，涵盖了从张量操作到模型训练的全过程。
- **Fast.ai**：以“自上而下”著称，非常适合想要快速上手实践的初学者。

### 深度阅读
- **李沐《动手学深度学习》(Dive into Deep Learning)**：中文界最优秀的深度学习教材，配套代码和视频。
- **Andrej Karpathy 的博客**：前 Tesla AI 总监，他的文章对理解神经网络底层逻辑极有帮助。
