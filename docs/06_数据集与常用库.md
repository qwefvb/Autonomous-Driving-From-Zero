# 6. 数据集与常用库

工欲善其事，必先利其器。本章汇总了自动驾驶算法开发中最常用的资源。

## 1. 主流数据集
- **nuScenes**：目前最流行的自动驾驶数据集，包含多摄像头、雷达点云、高精地图和完整的标注。
- **Waymo Open Dataset**：数据量极大、质量极高，是学术界和竞赛的首选。
- **KITTI**：自动驾驶数据集的鼻祖，虽然年代久远但依然是 3D 感知的经典入门集。
- **Argoverse 2**：侧重于轨迹预测和高精地图。

## 2. 常用代码库
- **感知端**：
  - [OpenMMLab (MMDetection3D)](https://github.com/open-mmlab/mmdetection3d)：涵盖了几乎所有主流的 3D 感知算法。
  - [OpenPCDet](https://github.com/open-mmlab/OpenPCDet)：专注点云检测。
- **仿真与规划端**：
  - [CARLA](https://carla.org/)：最流行的开源自动驾驶模拟器。
  - [HighwayEnv](https://github.com/eleurent/highway-env)：轻量级的决策与强化学习环境。
- **通用工具**：
  - [ROS (Robot Operating System)](https://www.ros.org/)：自动驾驶系统的“操作系统”。
  - [WandB (Weights & Biases)](https://wandb.ai/)：实验管理与可视化神器。
