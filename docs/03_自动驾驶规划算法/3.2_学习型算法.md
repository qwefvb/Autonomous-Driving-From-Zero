# 3.2 学习型算法

随着深度学习的发展，越来越多的研究尝试直接利用数据来学习“如何驾驶”，以应对复杂的长尾场景。

## 1. 模仿学习 (Imitation Learning)
通过模仿人类专家的驾驶数据来学习。
- **行为克隆 (Behavior Cloning)**：最直接的方法，但在遇到没见过的情况时容易发生偏移。
- **逆强化学习 (Inverse RL)**：学习专家的代价函数 (Cost Function)。

## 2. 强化学习 (Reinforcement Learning)
通过在仿真环境中不断试错，利用奖励机制 (Reward) 来优化驾驶策略。
- **经典算法**：PPO, SAC。
- **挑战**：模拟器与真实世界的 Gap，以及奖励函数的定义难度。
