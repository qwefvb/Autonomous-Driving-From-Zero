# 4.3 VLA (Vision-Language-Action)

VLA 模型是将大语言模型 (LLM) 的推理能力、视觉模型 (VLM) 的感知能力与机器人的控制动作 (Action) 相结合。

## 1. 核心概念
VLA 让车辆不仅能“看到”障碍物，还能通过自然语言理解复杂的交通规则和场景语义。例如：“这辆救护车在闪灯，我应该靠边停。”

## 2. 代表性工作
- **RT-2 (Robotics Transformer 2)**：Google 推出的通用 VLA 模型，虽然最初用于机械臂，但其思想正被快速引入自动驾驶。
- **Linguist**：利用语言指导自动驾驶行为。
