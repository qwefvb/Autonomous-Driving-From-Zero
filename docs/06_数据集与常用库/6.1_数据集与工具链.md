# 📦 6. 数据集与常用库

> [!NOTE]
> **工欲善其事，必先利其器。** 本章节汇总了自动驾驶算法开发中最核心的数据资源与开源工具链，建议将其作为日常开发的“工具手册”收藏。

---

## 🚀 1. 主流数据集 (Datasets)

| 数据集名称      | 核心优势                                | 适用场景                 |
| :-------------- | :-------------------------------------- | :----------------------- |
| **nuScenes**    | 全场景覆盖、包含多摄像头+LiDAR+高精地图 | 传感器融合、全栈算法验证 |
| **Waymo Open**  | 数据规模庞大、标注精度极高              | 工业级模型训练、学术竞赛 |
| **KITTI**       | 经典鼻祖级数据集、任务划分明确          | 3D 感知入门、视觉里程计  |
| **Argoverse 2** | 侧重于轨迹预测、高精地图细节丰富        | 预测、规划与地图学习     |

## 🛠️ 2. 常用代码库与工具链

### 2.1 深度学习核心框架与模型库

| 工具名称         | GitHub 仓库地址                                       | 核心说明                             |
| :--------------- | :---------------------------------------------------- | :----------------------------------- |
| **PyTorch**      | `https://github.com/pytorch/pytorch`                  | 行业标准深度学习框架，动态图调试极佳 |
| **Transformers** | `https://github.com/huggingface/transformers`         | 使用各种各样模型的核心库             |
| **Diffusers**    | `https://github.com/huggingface/diffusers`            | 扩散模型常用库                       |
| **Timm**         | `https://github.com/huggingface/pytorch-image-models` | 视觉网络架构及其预训练权重全集       |

| 工具名称         | GitHub 仓库地址                                | 核心说明                                       |
| :--------------- | :--------------------------------------------- | :--------------------------------------------- |

| **OpenCV**       | `https://github.com/opencv/opencv`             | 图像处理、几何变换与相机标定的“瑞士军刀”       |

## 🎯 建议练习

> [!TIP]
> **实践是检验真理的唯一标准：**
>
> 1. **数据初探**：下载 [nuScenes Mini](https://www.nuscenes.org/download) 数据集，尝试使用官方提供的 `nuscenes-devkit` 可视化一帧雷达点云与环视相机图像。
> 2. **环境搭建**：安装并启动 **CARLA** 仿真器，运行示例代码 `manual_control.py` 体验在仿真环境中的驾驶。
