# 📦 6. 数据集与常用库

> [!NOTE]
> **工欲善其事，必先利其器。** 本章节汇总了自动驾驶算法开发中最核心的数据资源与开源工具链，建议将其作为日常开发的“工具手册”收藏。

---

## 🚀 1. 主流数据集 (Datasets)

| 数据集名称          | GitHub 链接                                                   | 核心优势                                  | 适用场景                     |
| :------------------ | :------------------------------------------------------------ | :---------------------------------------- | :--------------------------- |
| **nuScenes**        | `https://github.com/nutonomy/nuscenes-devkit`                 | 全场景覆盖、包含多摄像头 + LiDAR + 高精地图 | 传感器融合、全栈算法验证     |
| **Waymo Open**      | `https://github.com/waymo-research/waymo-open-dataset`        | 数据规模庞大、标注精度高                    | 工业级模型训练、学术竞赛     |
| **KITTI**           | `https://github.com/bostondiditeam/kitti`                     | 经典鼻祖级数据集、任务划分明确              | 3D 感知入门、视觉里程计      |
| **Argoverse 2**     | `https://github.com/argoverse/av2-api`                        | 侧重轨迹预测，高精地图细节丰富              | 预测、规划与地图学习         |
| **BDD100K**         | `https://github.com/bdd100k/bdd100k`                          | 多天气、多时段、多任务统一标注              | 检测、分割、跟踪、驾驶场景理解 |
| **Lyft Level 5**    | `https://github.com/lyft/nuscenes-devkit`                     | 与 nuScenes 风格接近，便于多数据集迁移      | 多传感器感知、轨迹预测       |
| **SemanticKITTI**   | `https://github.com/PRBonn/semantic-kitti-api`                | 基于 KITTI 扩展的逐点语义标注               | 激光雷达语义分割、场景理解   |
| **Cityscapes**      | `https://github.com/mcordts/cityscapesScripts`                | 城市场景精细分割标注，社区使用广泛          | 语义分割、全景分割、视觉理解 |
| **nuImages**        | `https://github.com/nutonomy/nuscenes-devkit`                 | nuScenes 同生态的 2D 图像数据，标注细致      | 2D 检测、实例分割、场景理解  |
| **nuPlan**          | `https://github.com/HorizonRobotics/nuplan-devkit`            | 面向规划的闭环基准，场景切分与评测完整      | 规划、决策、闭环评测         |
| **NAVSIM**          | `https://github.com/autonomousvision/navsim`                  | 介于开环与闭环之间的伪仿真评测框架          | 端到端驾驶评测、规划验证     |
| **Bench2Drive**     | `https://github.com/Thinklab-SJTU/Bench2Drive`                | 面向闭环端到端驾驶的多能力基准，场景丰富    | 闭环评测、端到端驾驶、基准测试 |

## 🛠️ 2. 常用代码库与工具链

| :--------------- | :---------------------------------------------------- | :----------------------------------- |

## 🎯 建议练习

> [!TIP]
> **实践是检验真理的唯一标准：**
>
> 1. **数据初探**：下载 [nuScenes Mini](https://www.nuscenes.org/download) 数据集，尝试使用官方提供的 `nuscenes-devkit` 可视化一帧雷达点云与环视相机图像。
> 2. **环境搭建**：安装并启动 **CARLA** 仿真器，运行示例代码 `manual_control.py` 体验在仿真环境中的驾驶。
