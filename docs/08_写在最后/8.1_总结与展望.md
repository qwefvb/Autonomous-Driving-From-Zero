# 8. 写在最后

恭喜你！走到了这份指南的末尾。

## 核心心法

- **多动手**：代码跑通的一瞬间，抵得上读十遍论文。
- **重基础**：无论是 ResNet 还是 Transformer，核心原理往往是相通的。
- **保持好奇心**：关注行业动态，但不要盲目追逐每一个热点，但对每个热点最好都要有些了解。
- **积极拥抱AI**：这很关键，建议每半个月抽出一点时间专门用来研究AI进展，这会大大提高你的科研效率。

## 给新手的寄语

自动驾驶的愿景是“让出行更安全、更美好”。无论你是为了学术理想，还是为了工程落地，希望这份指南能为你指明方向。

如果你觉得这份项目对你有帮助，欢迎向身边的同学推荐，或者提交 Pull Request 贡献你的见解。

**路漫漫其修远兮，吾将上下而求索。**
